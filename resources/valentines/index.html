Valentines

More ways for players to connect

Valentines can help build friendships and make server community better together. Players can hug, kiss, like eachother, use moods that will be displayed on their profiles as well as stats, or use any effect.

And if you really like that one person, you can marry him and stay together until the server closes.

Note: This plugin was originally developed by Wejkey.
