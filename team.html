How to become a tester

Help me with development by testing early builds and providing honest feedback.

  1. 01

    Join the Discord

    Use Discord server or DMs and ask about this position.

  2. 02

    Introduce yourself

    Do you have experiences with testing plugins? Do you have personal/public server?

  3. 03

    Test builds

    Download latest builds from github or spigot and tell me your experiences.

  4. 04

    Report clearly

    Include the plugin version, server version, steps to reproduce the problem(if possible), and console errors when relevant.

Do you want to be part of the testing team?

  • Stay in the team after testing phase ends.
  • Ask for builds that are not public yet.
  • Promote your public server by using this plugin.

Tester access may be limited based on the current needs of the project.
