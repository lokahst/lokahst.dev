How to become a tester

Help test early builds and share clear feedback.

  1. 01

    Join the Discord

    Use Discord or DMs and ask about tester access.

  2. 02

    Introduce yourself

    Do you have experience testing plugins? Do you run your own server?

  3. 03

    Test builds

    Download the latest builds from GitHub or Spigot and share what you find.

  4. 04

    Report clearly

    Include plugin version, server version, reproduction steps, and any console errors.

Do you want to be part of the testing team?

  • Stay in the team after the initial testing phase ends.
  • Request access to pre-release builds.
  • Use the plugin on your public server to support your testing work.

Tester access may be limited based on the current needs of the project.
