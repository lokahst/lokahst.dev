Friends

This plugin contains Friends system which is a list of friends you added while playing. In main GUI click on Friends menu to manage your friend system if you have valentines.friends.use permission.

Manage friends

You can send request to another players by opening Friends menu, clicking on Add friend, and entering player's name in chat, if you have valentines.friends.add permission. The player will receive a friend request. This player can either accept or deny your request by opening Pending requests in Friends menu and selecting one of the options.

Remove your friends

In Friends menu, select one of your friends and click Remove friend. Friendship will be removed for both players.
